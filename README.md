# -API-M-nima-com-ASP.NET-Core
Objetivo:
Construir do zero uma API RESTful mínima em ASP.NET Core para demonstrar na prática como serviços
se comunicam em uma arquitetura distribuída. Ao final, os alunos terão um serviço HTTP funcional com
CRUD completo, testável pelo Swagger, e entenderão como esse padrão se encaixa no contexto de
microsserviços
